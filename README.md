# LLR-site
LLR  Hall Website
Deployed on Heroku

# Installation and Setup


- Create a virtual environment

      `python3 -m venv env`
- Activate it

      `source env/bin/activate` (For unix like OS)

      `source env\Scripts\activate` (For windows)

- Install requirements

      `pip3 install -r requirements.txt`

- Run it 

      `flask run`
